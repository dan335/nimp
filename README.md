# NIMP

https://nimp.app

Node-Based Image Manipulation Program

![](https://nimp.app/static/screenshots/screenshot03.JPG)

### Run
```
start Mongodb
git clone https://github.com/dan335/nimp
cd nimp
npm install
npm run dev
View in browser at http://localhost:3000
```
